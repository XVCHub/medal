Python Example
import base64, requests encoded = base64.b64encode(open("script.luac", "rb").read()) r = requests.post("https://medaldecompile.pages.dev/decompile", data=encoded) print(r.text)

WS /decompile_ws WebSocket · async batch decompile with message IDs
Send (JSON)
fieldtypedescription

Receive (JSON)
fieldtypedescription
idstringsame id you sent

JavaScript Example
const ws = new WebSocket("wss://medaldecompile.pages.dev/decompile_ws") ws.onopen = () => ws.send(JSON.stringify({ id: "job_1", encoded_bytecode: btoa(String.fromCharCode(...bytecodeBytes)) })) ws.onmessage = e => { const { id, decompilation } = JSON.parse(e.data) console.log(id, decompilation) }
Notes
ℹ Base64 required. always encode bytecode to base64 before sending — raw bytes will be rejected.
ℹ Luau only. lua 5.1 .luac bytecode format is different — make sure your file is compiled luau.
ℹ WebSocket for bulk. /decompile_ws lets you send multiple jobs at once, each identified by id.
